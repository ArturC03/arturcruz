CSS é uma máquina de estados?

Muitas pessoas olham para o CSS apenas como uma forma de "pintar" o HTML. Mas quando começas a olhar para as pseudo-classes como :hover, :active, ou :focus-within, percebes que estás a definir transições entre estados de um sistema visual.

Em Ciência de Computadores, estudamos Autómatos Finitos. Um botão no teu site é um pequeno autómato: tem o estado Idle, o estado Hover e o estado Pressed. O CSS é a linguagem que descreve como o sistema se comporta em cada um desses estados.

Pensar desta forma torna o debug de estilos muito mais lógico. Se algo está mal, não é apenas um "erro de cor", é um erro de transição de estado.
